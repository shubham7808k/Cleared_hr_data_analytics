# 📈HR Data Analytics Dashboard

## ♦️Overview

This project leverages IBM Cognos Analytics to transform raw data into actionable business intelligence. By utilizing advanced reporting and interactive dashboard, the project identifies key trends, optimizes data exploration, and allows the HR Department to take the appropriate steps to keep the organization running smoothly and profitably.

---

## 📝Objectives

- **Data Modeling:** Creating a robust data module to ensure accurate reporting.
- **Interactive Visualization:** Design an intuitive dashboard for easy data exploration.
- **Business Insights:** Extract high-impact findings to support data-driven decision-making.
- **Technical Mastery:** Demonstrate proficiency in IBM Cognos tools, including calculations, filters, and conditional formatting.
---

## 📂 Dataset Profile

- **Name:** HR Analytics Data
- **Source:** https://www.kaggle.com/datasets/hopesb/hr-analytics-dataset
- **Format:** CSV
- **Dimensions:** 2846 rows * 28 columns
- **Domain:** Human Resource Management
  
**Description:**
This dataset contains detailed information about employees, departments, training programs, and other HR-related metrics. This project helps us with interpreting organizational data and finding the people-related trends in the data.

---
## 📊Dashboard Features
- KPIs such as Total Employee Count, Total Job Profiles, Total Divisions.
- Peformance Score by Department Type
- Engagement Score by Job Title
- Employee Count by Job Title
- Satisfaction Score by Pay Zone
- Employee Count by Employee Type
---
## 📍Key Insights

### Trend and Categorical Analysis :
- Executive Office and IT/IS Departments have the highest Performance Score at 4.0.
- Job titles: Data Architect and Enterprise Architect have the highest Engagement Score at 4.0.
- PayZone Zone B has the highest Satisfaction Score at 3.06, followed by Zone C at 3.03.
- There are maximum number of Full-Time employees followed by Contract and then Part-Time ones.

### Anomaly Detection :
- Software Engineering Department has the lowest Performance Score at 1.0 indicating for improvements in its process management.
- Production Technician I has maximum employees counting to 1241, followed by Production Technician II with 486, suggesting that these roles require most amount of manual labour.

### Prediction :
- Based on current data, it is projected that by 2023-10-18, Zone A will exceed Zone C in Satisfaction Score by 0.02.
